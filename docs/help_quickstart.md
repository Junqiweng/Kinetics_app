# 快速上手（建议按这个顺序）

## 一次完整工作流

1. **选模型**：左侧选择 `反应器(PFR/CSTR/BSTR)` 与 `动力学(power_law / langmuir_hinshelwood / reversible)`
2. **填反应网络**：输入物种名、反应数，填写化学计量数矩阵 $\nu$（反应物负、生成物正）
3. **给初值**：填写 $k_0$、$E_a$、反应级数 $n$（以及 L-H / 可逆的额外参数），勾选要拟合的参数
4. **准备 CSV**：先下载模板，再把实验数据粘进去（列名必须严格一致）
5. **上传 CSV**：在「② 实验数据」上传
6. **选拟合目标**：选择 `Fout/Cout/X`，并勾选“进入目标函数”的物种
7. **开始拟合**：到「③ 拟合与结果」点 `🚀 开始拟合`
8. **看诊断**：奇偶校验图 + 残差 + 剖面（PFR: 随 V；BSTR: 随 t；CSTR: 随 t 逼近稳态）
9. **导出复现**：导出参数/对比表/图像，并导出配置 JSON

## 新手最稳的默认策略

- 先用示例数据跑通流程，再换自己的数据
- 拟合不动时：
  - 把 `diff_step` 调到 `1e-2 ~ 1e-3`
  - 开启 `Multi-start`（Start Points 5~20）
  - ODE 求解器切到 `BDF` 或 `Radau`

## 你应该先确认的 3 件事（避免白忙）

- 物种名与你 CSV 的列名一致（例如 `A` 对应 `F0_A_mol_s`）
- 你选择的“拟合目标变量 + 目标物种”对应测量列 **没有 NaN/空值/非数字**
- 你的单位体系一致：温度 K，体积 m³，流量 m³/s，浓度 mol/m³，摩尔流量 mol/s
